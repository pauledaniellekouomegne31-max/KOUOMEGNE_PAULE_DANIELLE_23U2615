# CoiffData - Analyse du secteur de la coiffure au Cameroun 

Projet universitaire réalisé dans le cadre du cours **INF 232**. 
Cette application permet de collecter des données statistiques sur les habitudes des clients dans les salons de coiffure et propose un tableau de bord analytique pour les administrateurs.

Liens du projet
- Site web (Vercel) : [https://kouomegne-paule-danielle-23-u2615-x.vercel.app/]  , (https://kouomegne-paule-danielle-23-u2615-x.vercel.app/)
- Base de données : Supabase

 Technologies utilisées
- Frontend : HTML5, CSS3 (Modern Design)
- Backend : Python (Flask)
- Analyse de données : Pandas
- Base de données : Supabase (PostgreSQL)
- Déploiement : Vercel

 Fonctionnalités
- Formulaire dynamique : Collecte d'informations (type de coiffure, prix, quartier, notes de satisfaction).
- Dashboard Admin : Visualisation des statistiques en temps réel (prix moyen, évolution des visites, radar des notes).
- Export de données : Possibilité de télécharger les réponses au format CSV.

 Accès Admin
Pour accéder au tableau de bord :
1. Aller sur la page `/login`.
2. Utiliser le mot de passe : `admin2026`

Structure du projet
- `/api/index.py` : Serveur Flask principal.
- `/static/` : Fichiers CSS et images.
- `/templates/` : Pages HTML (accueil, formulaire, dashboard).
- `vercel.json` : Configuration pour le déploiement Cloud.
